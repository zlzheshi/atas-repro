## 完整 ImageNet 训练

等待 4 张 GPU 空闲后自动启动：

```bash
nohup bash scripts/run_full_imagenet_author_wait.sh \
  > /mnt/t1b6/xuzhejia/logs/atas_full_author_wait.log 2>&1 &
```

直接 4 卡启动：

```bash
CUDA_VISIBLE_DEVICES=0,1,2,3 PYTHONPATH=. \
python -m torch.distributed.run \
  --standalone \
  --nproc_per_node=4 \
  train_atas.py \
  --config configs/atas_vitb_imagenet_full_author.yaml \
  --data-root /mnt/t1b6/xuzhejia/datasets/imagenet/train
```

从 checkpoint 恢复：

```bash
CUDA_VISIBLE_DEVICES=0,1,2,3 PYTHONPATH=. \
python -m torch.distributed.run \
  --standalone \
  --nproc_per_node=4 \
  train_atas.py \
  --config configs/atas_vitb_imagenet_full_author.yaml \
  --data-root /mnt/t1b6/xuzhejia/datasets/imagenet/train \
  --resume outputs/atas_vitb_imagenet_full_author/checkpoint_epoch_5.pt
```

查看训练日志：

```bash
tr '\r' '\n' < /mnt/t1b6/xuzhejia/logs/atas_full_author_wait.log | tail -n 30
```

查看 checkpoint：

```bash
ls -lh outputs/atas_vitb_imagenet_full_author/checkpoint_epoch_*.pt
```
