# ATAS 完整复现实验方案与当前进展

## 复现目标

复现 ICCV 2025 论文 **ATAS: Any-to-Any Self-Distillation for Enhanced Open-Vocabulary Dense Prediction** 的核心思想：使用冻结的原始 CLIP image encoder 作为 teacher，通过自蒸馏训练 student image encoder，使其同时保持全局语义能力并增强局部 patch 表征。

课程项目中采用“核心方法复现 + 子集定量评估 + patch 级定性可视化 + 消融实验”的路线。

## 已完成的工程内容

1. 环境与数据
   - 服务器：4 x RTX A6000。
   - Conda 环境：`atas`。
   - 完整 ImageNet train 已解压。
   - 构造 ImageNet-100x200 子集，共 20,000 张图像。

2. 训练代码
   - 实现 Mosaic 构造。
   - 实现 GLD、LLD、GGD 三个核心 loss。
   - 支持 AMP、checkpoint、resume。
   - 修复 OpenCLIP ViT token 提取路径，使其与 `encode_image()` 对齐。

3. 稳定训练
   - 配置：`configs/atas_vitb_subset_100x200_stable.yaml`
   - 训练：已从 3 epochs 继续到 6 epochs。
   - checkpoint：`outputs/atas_vitb_subset_100x200_stable/checkpoint_epoch_6.pt`

4. 轻量评估
   - 使用 100 类子集，每类 160 张 gallery、40 张 query。
   - 指标：Top-1 kNN、Top-5 neighbor hit、centroid margin。

## Full ATAS 当前结果

| 模型 | Top-1 kNN | Top-5 Hit | Centroid Margin |
| --- | ---: | ---: | ---: |
| OpenCLIP baseline | 0.8900 | 0.9653 | 0.0757 |
| ATAS epoch 3 | 0.8913 | 0.9618 | 0.2239 |
| ATAS epoch 6 | 0.8965 | 0.9650 | 0.2252 |

结论：

- Top-1 kNN 在 epoch 6 提升到 0.8965，说明训练没有破坏 CLIP 的语义判别能力，并在当前子集上带来一定提升。
- Centroid margin 明显提升并保持稳定，说明子集上类间分离更清晰。
- 同图 cosine drift 表明 checkpoint 不是简单复制原始 CLIP，而是发生了实际特征调整。

## 消融实验结果

论文核心由三个 loss 组成，因此课程复现中设计如下消融：

| 实验 | GLD | LLD | GGD | Top-1 kNN | Top-5 Hit | Centroid Margin |
| --- | --- | --- | --- | ---: | ---: | ---: |
| OpenCLIP baseline | 否 | 否 | 否 | 0.8900 | 0.9653 | 0.0757 |
| GLD only | 是 | 否 | 否 | 0.2412 | 0.5070 | -0.0069 |
| GLD + LLD | 是 | 是 | 否 | 0.2117 | 0.4542 | -0.0152 |
| Full ATAS | 是 | 是 | 是 | 0.8913 | 0.9618 | 0.2239 |

结果说明：

- `GLD only` 和 `GLD + LLD` 明显损伤图像级语义判别能力，说明只强化局部对齐会破坏 CLIP 原有全局语义结构。
- 加入 `GGD` 后，Full ATAS 基本保持 OpenCLIP 的 Top-1 kNN，同时显著提升 centroid margin。
- 该消融趋势支持论文中的设计动机：GLD/LLD 负责增强局部表征，GGD 用来稳定全局语义能力。

消融输出：

- `outputs/ablation_summary/ablation_metrics.md`
- `outputs/eval_subset_100x200_gld_only_knn/metrics.json`
- `outputs/eval_subset_100x200_gld_lld_knn/metrics.json`

## VOC2012 零样本分割补充实验

已接入 PASCAL VOC2012 val split，使用 ViT patch token 与 20 个前景类别文本特征直接匹配，作为轻量级 zero-shot dense prediction 代理评估。

| 模型 | 前景 mIoU | 前景像素准确率 | 平均类别准确率 |
| --- | ---: | ---: | ---: |
| OpenCLIP baseline | 0.4016 | 0.5565 | 0.5680 |
| ATAS epoch 3 | 0.3604 | 0.5174 | 0.5789 |
| ATAS epoch 6 | 0.3287 | 0.4854 | 0.5462 |

该结果说明：epoch 6 在 ImageNet 子集 kNN 上更好，但没有直接提升 VOC2012 简化 patch-text 分割指标。汇报中应把 VOC2012 作为标准数据集补充实验与局限性分析，而不是主要正向结果。
